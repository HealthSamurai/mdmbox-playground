# Agent Guidance

For Aidbox and MDMbox behavior, configuration, APIs, and examples, use the
official Health Samurai documentation as the source of truth:

- Aidbox: https://www.health-samurai.io/docs/aidbox
- MDMbox: https://www.health-samurai.io/docs/mdmbox

When working on this repository, agents should check those docs before relying
on memory or assumptions about Aidbox or MDMbox. If local examples disagree
with the docs, treat the docs as authoritative and call out the discrepancy.
